An output of one means that the value is not accepted. A 0 means that it is accepted.


expected outputs for each value entered: 
input: 100 
output: 0 (rejected)

input: 1010
output: 1 (accepted)

input: 1 
output: 0 (rejected)

input: 0010
output: 1 (accepted)

input: 0011001
output: 0 (rejected)
